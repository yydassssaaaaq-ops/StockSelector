# backtest

当前包含第一条最小真实历史回测闭环：

- `engine.py`：横截面选股回测。信号使用信号日收盘后可得日线字段，默认下一交易日收盘成交，Top N 等权持有到下一次调仓成交日。
- `metrics.py`：集中计算累计收益、年化收益、最大回撤、年化波动率、夏普比率、胜率、单期收益、换手率、交易次数、有效期数和相对基准超额收益。

当前默认命令：

```powershell
python scripts\run_minimal_backtest.py
```

已知限制：

- 股票池仍来自当前真实快照或 CSV，历史退市股票未完整纳入，存在幸存者偏差。
- 腾讯历史源缺少换手率、主力资金和估值字段，评分通过动态权重降置信处理。
- 停牌、涨跌停可成交性和真实盘口滑点未完整建模。

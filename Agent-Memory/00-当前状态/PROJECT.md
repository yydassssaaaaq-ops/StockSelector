# PROJECT

- 项目名称：StockSelector / A股智能选股系统
- 项目定位：面向后续 A 股智能选股系统开发的工程仓库。
- 核心目的：建立可持续迭代、可记录、可验证的项目闭环基础设施。
- 当前主线：先建立 Agent-Memory 和工程壳，再由用户确认业务事实。
- 当前阶段：闭环基础设施初始化。
- 已确认事实：项目英文名、中文名、当前任务和当前不开发具体选股算法。
- 待确认事项：数据源、复权方案、股票池、指标、筛选规则、回测框架、模型类型、Agent 决策方式、数据库、实盘接口、交易周期、输出排序规则。
- 长期有效原则：不把推测写成事实，不写入真实密钥，不自动 Commit 或 Push。

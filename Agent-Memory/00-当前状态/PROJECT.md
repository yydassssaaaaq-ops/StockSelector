# PROJECT

- 项目名称：StockSelector / A股智能选股系统
- 项目定位：面向 A 股选股研究系统开发的工程仓库。
- 核心目的：建立可持续迭代、可记录、可验证、时间点可信的真实数据选股研究闭环。
- 当前主线：在既有 Agent-Memory 和历史案卷库基础上，逐步建设真实行情、实时扫描、历史研究、策略验证、组合构建和解释能力。
- 当前阶段：第一条真实行情选股业务链已具备实时快照筛选；历史回测已升级为 `historical_ohlcv_v1` 可信研究底座。
- 已确认事实：项目英文名、中文名、当前任务；默认可使用新浪市场中心 `hs_a` 行情快照跑通规则筛选报告；可使用腾讯前复权日 K 跑通 point-in-time 历史验证并与沪深 300、股票池等权和单因子基线比较。
- 待确认事项：无幸存者偏差历史股票池、东财历史源恢复后的字段完整回测、行业/财务/资金流因子、Agent 决策方式、数据库、实盘接口、交易周期、正式组合构建和风险约束。
- 长期有效原则：不把推测写成事实，不写入真实密钥；任务完成、测试通过且工作区无明显异常后，可在当前分支正常 Commit 并 Push，禁止 force push。

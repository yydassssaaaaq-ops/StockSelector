# PROJECT

- 项目名称：StockSelector / A股智能选股系统
- 项目定位：面向 A 股智能选股系统开发的工程仓库。
- 核心目的：建立可持续迭代、可记录、可验证的真实数据选股闭环。
- 当前主线：在既有 Agent-Memory 和历史案卷库基础上，逐步建设真实行情、规则筛选、报告、回测和解释能力。
- 当前阶段：第一条真实行情选股业务链已具备快照筛选和最小历史回测闭环。
- 已确认事实：项目英文名、中文名、当前任务；默认可使用新浪市场中心 `hs_a` 行情快照跑通规则筛选报告；可使用腾讯前复权日 K 跑通最小历史回测并与沪深 300 基准比较。
- 待确认事项：无幸存者偏差历史股票池、东财历史源恢复后的字段完整回测、行业/财务因子、Agent 决策方式、数据库、实盘接口、交易周期、正式输出排序规则。
- 长期有效原则：不把推测写成事实，不写入真实密钥；任务完成、测试通过且工作区无明显异常后，可在当前分支正常 Commit 并 Push，禁止 force push。
